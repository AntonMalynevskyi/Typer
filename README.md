Typer for Kyrenti. Automated logging in, doing tasks and going to the next one. Should also work with english and IT courses

To use either download .exe from releases or use commands from below


To start:
```
git clone https://github.com/AntonMalynevskyi/Typer.git
cd Typer
pip install -r requirments.txt
python main.py
```

To end the script just close the terminal with it or the browser where it works
